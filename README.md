# Cash Flow Tracker

## Overview

Cash Flow Tracker is a personal finance management web application that helps users track their salary, expenses, and remaining balance. The application provides real-time financial insights through dynamic calculations and visualizations.

## Features

* Add and manage salary information
* Add and delete expenses dynamically
* Real-time remaining balance calculation
* Expense vs Remaining Balance pie chart visualization
* Low balance warning alert system
* Data persistence using Local Storage
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js
* Local Storage API

## How It Works

1. Enter your salary.
2. Add expenses with name and amount.
3. Track your remaining balance automatically.
4. View spending distribution through a pie chart.
5. Receive warnings when the remaining balance falls below the defined threshold.
6. Refresh the page without losing saved data.
